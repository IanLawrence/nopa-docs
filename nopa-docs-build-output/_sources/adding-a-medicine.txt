*****************
Adding a medicine
*****************

After the patient has been added a logical next step would be to add a medicine. On the **Participants** tab click the patients name which brings up the contextual interface seen below

.. figure::  _images/Server/8.png
   :align:   center
   :scale: 75

   The Participants contextual interface

Click on the **Meds** box which will show a list of the medicines for that patient

.. figure::  _images/Server/9.png
   :align:   center
   :scale: 75

   Medicine List

Clicking **+medicine** will open the interface to add a new medicine. Here the doctor can specify the name, the medicine alias, the days of the week it will be taken as well as how often and how many doses should be taken

.. figure::  _images/Server/10-1.png
   :align:   center
   :scale: 75

   Add Medicine

Saving this automatically updates the client with the new information

.. figure::  _images/Client/1-1.jpg
   :align:   center
   :scale: 75

   Next Meds List
